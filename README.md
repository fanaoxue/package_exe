# klbhyb-electron

> An electron-vue project

## Build Setup

### 安装 vue-cli 和 脚手架样板代码

```js
npm install -g vue-cli

vue init simulatedgreg/electron-vue my-project
```

### 安装依赖并运行你的程序

> 安装依赖建议使用yarn

```js
cd my-project

yarn #

yarn run dev
```
